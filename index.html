Icons
Phosphor Bold is the default icon set for all UI icons. Bold weight for default and hover states, Fill weight for active/selected states. Wire custom icons are used only where Phosphor has no equivalent — listed below as they are introduced into the system.
Icon sets
SetWeightUsageHow to use
Phosphor Bold · Fill All standard UI icons <i class="ph-bold ph-{name}"></i>
<i class="ph-fill ph-{name}"></i>
Wire custom Inline SVG Product-specific concepts with no Phosphor match Inline SVG · fill="currentColor" · sized via CSS
Active state rule
Nav rail and any toggleable icon: Bold = default/hover · Fill = active/selected. This is the only place Fill weight is used — all other contexts use Bold only.
