# Wire Design System

Design system documentation and interactive prototype for Wire's multiplatform app.

| | |
|---|---|
| **Docs** | [wolfjwire.github.io/wire-design-system](https://WolfiWire.github.io/wire-design-system/) |
| **Prototype** | [wolfjwire.github.io/wire-design-system/prototype.html](https://WolfiWire.github.io/wire-design-system/prototype.html) |

## Stack
- Base: shadcn/ui, Maia style (radius 12px)
- Font: Inter (all platforms)
- Icons: Phosphor Bold + Wire custom
- Accent: 6-color user-selectable system
- Modes: Light + Dark

## Files
- `index.html` — WDS Component Reference (tokens, components, specs)
- `prototype.html` — Fullscreen interactive app prototype
